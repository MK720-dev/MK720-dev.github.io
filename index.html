Research Interests & Aspirations

I work at the intersection of robust machine learning and optimization, with a focus on understanding how learning systems behave in real-world, high-stakes settings. My goal is to help bridge the gap between ML theory and practical deployment in domains where reliability matters.

As such, my research agenda focuses on the mathematical and algorithmic foundations of machine learning, with emphasis on the following themes:

  • Optimization-Aware Understanding of Neural Network Dynamics: Developing theoretical and computational tools to study learning trajectories, stability, curvature, and convergence behavior in high-dimensional nonconvex systems.
  • Robust and Deployable Machine Learning for High-Stakes Domains: Designing learning systems that maintain reliability under noise, distribution shift, and real-world variability, with applications to human health, rehabilitation technology, and distributed robotic systems.
  • Advancing AutoML and Architecture Search: Improving automated techniques for neural architecture design, hyperparameter exploration, and model evaluation using insights from optimization dynamics, generalization theory, and empirical training behavior.

Graduate Coursework & Research Experience

Undergraduate Researcher — CardioSegNet (Phase 1)

I2S, University of Kansas — Jan 2023 to May 2025

U-Net Biomedical ML HDF5 tf.data MRI Processing Dash/Plotly
  • Built CardioSegNet, a lightweight U-Net–based system for automatic left-ventricle (LV) segmentation on cardiac MRI slices using the ACDC dataset.
  • Designed and implemented the baseline 2D U-Net architecture (encoder–decoder, skip connections, BCE + Dice loss) optimized for fast LV segmentation.
  • Engineered a full MRI preprocessing pipeline: HDF5 parsing, slice extraction, intensity normalization, resizing, ROI filtering, and binary LV mask generation.
  • Developed a high-performance tf.data input pipeline using parallel reads, caching, shuffling, and prefetching to accelerate training.
  • Achieved strong baseline accuracy (~0.82 Dice) through systematic experimentation, hyperparameter tuning, and quantitative evaluation.
  • Built a custom interactive MRI viewer using Dash & Plotly for visualizing model predictions, ground truth masks, and slice-by-slice overlays.
  • Conducted qualitative error analysis (apex/base failures, contour smoothness issues) to guide Phase 2 architectural improvements.
  • Documented architectural rationale, training decisions, and evaluation methodology to enable reproducibility and future AutoML experimentation.

MATH 796 — Optimization & Machine Learning

Graduate Course Project Work

Optimization Loss Landscapes PCA FastAPI React/Vite AdamW U-Net
  • Designed OptimizerArena, an interactive system for exploring high-dimensional neural-network loss landscapes using PCA-based dimensionality reduction and surrogate modeling.
  • Implemented gradient-based and quasi-Newton algorithms (SGD, Mini-batch GD, Adam, RMSProp, BFGS) from first principles to analyze optimization dynamics.
  • Integrated training-trajectory visualization and loss-surface reconstruction to study curvature, saddle points, and convergence behavior across tasks.
  • Built a full-stack implementation: a FastAPI backend for training/PCA computations and a React + Vite frontend for real-time 3D visualization.
  • Investigated optimizer sensitivity to initialization, curvature, and dataset structure using cross-validation and trajectory diagnostics.
  • Developed an extensible experimentation framework supporting synthetic, regression, and classification datasets with user-defined architectures.
  • Extending CardioSegNet by developing a deeper and more robust U-Net architecture incorporating residual blocks, attention modules, and higher-resolution inputs.
  • Studying advanced optimization strategies (AdamW, cosine decay, warm restarts) to reduce instability and improve convergence on medical image segmentation tasks.
  • Experimenting with alternative activation functions (LeakyReLU, GELU, Swish/Mish) and regularization methods to improve robustness to noise and anatomical variability.
  • Designing a flexible framework for multi-class segmentation (LV, RV, Myocardium) while maintaining LV-only training for controlled experimentation.
  • Building infrastructure for a future AutoML agent that evaluates optimization dynamics and recommends architectural/hyperparameter improvements.
  • Studying how depth, receptive field size, skip-connection structure, and attention mechanisms affect generalization, boundary sharpness, and apex/base consistency.
